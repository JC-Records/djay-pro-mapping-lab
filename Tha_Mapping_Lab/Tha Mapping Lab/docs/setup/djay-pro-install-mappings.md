# Installing djay pro Mapping Files

## macOS

1. Download the `.djayMidiMapping` file.
2. Open **djay pro**.
3. Go to **MIDI > Configure**.
4. Click **Import Mapping** and select the file.
5. Restart djay if it does not appear immediately.

## Windows

1. Download the `.djayMidiMapping` file.
2. Open **djay pro** for Windows.
3. Open the **MIDI** configuration panel.
4. Use the import option to load the file.
5. Restart djay if needed.

## iOS (iPad / iPhone)

1. Save the mapping into the **Files** app.
2. Open **djay**.
3. Go to MIDI settings.
4. Tap **Import** and pick the file.
